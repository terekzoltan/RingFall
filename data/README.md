# Ringfall Generated Data

This folder is the local home for generated Ringfall run artifacts.

Keep `data/.gitkeep` tracked so the directory exists in fresh checkouts. Do not commit generated run outputs, replay dumps, traces, eval artifacts, temporary files, or local/private data. The generated subfolders are ignored in the repository `.gitignore`.

Expected local-only generated paths:

- `data/runs/`
- `data/replays/`
- `data/traces/`
- `data/eval/`
- `data/tmp/`
