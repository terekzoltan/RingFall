# Ringfall Schema Groups

This directory groups future Ringfall JSON Schema files. W1-S1 creates the group layout only and does not define real schemas.

Group purposes:

- `state/`: future world and state export schemas.
- `packets/`: future execution-impacting typed packet schemas.
- `traces/`: future run, cognition, action, and state-diff trace schemas.
- `memory/`: future memory, belief, and claim trace schemas.
- `eval/`: future evaluation event and summary schemas.
- `config/`: future exported configuration schema surfaces.

No `*.schema.json` files belong in W1-S1. Schema bodies, fields, examples, and validation tooling are deferred to later gated Wave 1 steps.
