# RingFall Design Docs

This folder contains the primary RingFall design canon and subsystem architecture documents.

Start with `Ringfall-First-Playable-Slice-v01.md` and `Ringfall-Architecture-and-Repo-Plan-v01.md` when preparing implementation work.

`Formal-Intervention-Gates-Refinery.md` records the approved post-Wave-1 direction for using Refinery as a bounded formal intervention gate. It is not a runtime implementation and does not approve a full formal world model.
